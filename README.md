<a id="introductions"></a>
## 🎯 项目介绍

**UCut** 是一款基于 AI 的视频智能处理工具，集**视频剪辑、视频生成、视频优化**三大核心能力于一体，现新增**在线视频下载**功能，覆盖从获取素材到成片输出的完整工作流。

只需点点鼠标，即可调用最前沿的人工智能模型完成复杂的视频处理任务。无需专业剪辑技能，人人都能创作高质量视频作品。

<a id="features"></a>
## 🚀 功能介绍

UCut 目前已支持 **13 种视频处理方法**，覆盖视频处理全链路。

### 🎬 视频剪辑

<details>
<summary><strong>人脸剪辑</strong> — AI 自动识别并裁剪人脸画面</summary>
<p>UCut 拥有神奇的人脸识别技术，能自动从视频中捕捉每一个人脸瞬间。轻轻一点，选择关注的人脸，UCut 会像魔法一样自动裁剪出相关片段，让你的主角时刻闪耀！</p>
</details>

<details>
<summary><strong>语音剪辑</strong> — 智能识别语音，精准裁剪</summary>
<p>智能识别视频中的语音内容并清晰展示。选择感兴趣的语音部分，UCut 精准裁剪出你想要的内容。</p>
</details>

### 🎨 视频生成

<details>
<summary><strong>口播生成</strong> — 一张图 + 文字 = 有声视频</summary>
<p>上传带有人脸的图像，选择喜欢的音色，输入文字，UCut 瞬间生成对应语音口型的有声视频。</p>
</details>

<details>
<summary><strong>字幕生成</strong> — 语音转文字，多字体多颜色</summary>
<p>智能识别音视频内容自动生成字幕，支持多种字体和颜色，与视频完美同步。</p>
</details>

<details>
<summary><strong>色彩生成</strong> — 黑白视频一键上色</summary>
<p>一键式黑白视频上色，让陈旧视频焕发绚丽色彩，经典重现生机！</p>
</details>

<details>
<summary><strong>音频生成</strong> — 图文转视频</summary>
<p>为静态图和文字赋予生命，转化为生动的视频。图像 + TTS 语音，创意无限延伸。</p>
</details>

<details>
<summary><strong>语言更换</strong> — 视频语音一键换语言</summary>
<p>将视频中的语音瞬间转换成你想要的任何语言，跨越语言障碍，与国际接轨。</p>
</details>

<a id="getting-started"></a>
## 🎯 快速入门

### 方式一：使用工程（推荐）
1. 在主界面点击 **新建**，填写工程名称和位置
2. 通过 **文件 → 导入** 或 **网上下载** 添加视频
3. 选择处理方法，按照提示使用

### 方式二：直接打开视频（✨ 新功能）
1. 主界面点击 **打开**，选择视频文件（支持 mp4/avi/mov/mkv/webm/flv 等格式）
2. 即可直接进入编辑界面，无需创建工程

### 方式三：在线下载后自动编辑
1. 点击 **网上下载**，粘贴视频链接
2. 勾选 **下载后直接打开视频（不创建工程）**
3. 下载完成后自动进入编辑
